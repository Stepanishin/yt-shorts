Я хочу создать приложение только для одного пользователя. Это должен быть полный цикл генерации видео shorts для youtube.

Технологии:
next.js + tailwind + typescript + mongoDb

1) Необходима первичная авторизация пользователя по паролю который будет храниться в .env файле.

2) Весь сайт должен быть на русском языке.

3) Shorts будут для испаноязычной аудитории.

4) Используй OpenAi Api. Ключ будем брать из .env файла.

5) Используй mongoDb. Ключ будем брать из .env файла.

6) Тема видео - анекдоты на испанском языке.

7) Шаблон видео - 9:16 (вертикальный формат для YouTube Shorts).

7.1) Примеры стиля:
- https://www.youtube.com/shorts/s1Z2i1J6BrE
- https://www.youtube.com/shorts/hvlM_ksrW9U

8) Я предоставлю 1 или несколько сайтов с анекдатоми на испанском языке. Нужно брать один не длинный анекдот который главное не повторялся у меня и превратить его в видео shorts.

9) Для каждого видео нужно генерировать яркий бекграунд с изображением природы который должен быть анимированный.

10) На бекграунд накладывать текст анекдота.

11) Под текст Анекдота накладывать рандомный смеющийся емодзи с права от текста и анимированный.

12) Генерировать фоновую музыку разную для каждого видео (веселая, upbeat).

13) из всего этого нужно сгенерировать видео и если возможно то выгрузить сразу даже на youtube если API это позволяет.

УТОЧНЕНИЯ:
- Платформа: Vercel (без Redis, т.к. один пользователь)
- Хранилище: Cloudflare R2
- Рендеринг видео: Remotion Lambda (AWS)
- Музыка: YouTube Audio Library (50-100 треков)
- Озвучка: НЕ нужна (только текст)
- Объем: 10-20 видео/день
